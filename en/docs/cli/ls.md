<p class="lead">List installed packages.</p>

##### `yarn ls` <a class="toc" id="toc-yarn-ls" href="#toc-yarn-ls"></a>

```sh
yarn ls
```

```sh
yarnpkg ls vx.x.x
├─ package-1@1.3.3
├─ package-2@5.0.9
│  └─ package-3@^2.1.0
└─ package-3@2.7.0
```
